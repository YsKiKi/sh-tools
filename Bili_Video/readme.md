## 功能
将A+1站缓存的M4S文件对合并为MP4视频文件。

## 使用方法

### 1. 准备文件
- 在脚本同级目录创建 `input` 文件夹
- 将缓存的视频文件夹如 `1234567890` 复制到 `input` 内

### 2. 运行脚本
```bash
python start.py
```

## 文件结构示例
```
项目目录/
├── start.py
├── input/
│   ├── 视频1/
│   │   ├── xxx-1.m4s
│   │   └── xxx-2.m4s
│   └── 视频2/
│       ├── yyy-1.m4s
│       └── yyy-2.m4s
├── output/      （自动生成）
└── temp/        （自动生成）
```

## 注意事项
1. **FFmpeg必需**：确保已安装FFmpeg并加入系统环境变量
2. **输出位置**：生成的MP4文件位于`output`文件夹
